# Juego de Memoria

- **Estado del juego**: Un objeto que contiene el estado actual del juego, incluyendo las cartas seleccionadas y el puntaje.
- **Acciones**:
    - Seleccionar carta
    - Comparar cartas
    - Reiniciar juego.

Aquí, el estado del juego sería más complejo, ya que incluiría información sobre las cartas seleccionadas y el progreso del juego. Una función específica para el juego manejará acciones como seleccionar cartas, compararlas y actualizar la puntuación.
